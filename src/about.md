안녕하세요, **데브고래밥**입니다.

이곳은 제가 직접 겪고, 고민하고, 해결한 것들을 기록하는 공간입니다. 공부 노트나 단순한 메모가 아니라, 한 편 한 편 공을 들여 쓴 글만 올립니다. 양보다 질이라고 믿기 때문입니다.

글 하나를 쓸 때마다 "이 글이 누군가에게 실제로 도움이 될까"를 먼저 생각합니다. 그 기준을 통과한 것들만 여기에 남깁니다.

## 주로 다루는 것들

- TypeScript · JavaScript
- SvelteKit · React
- Node.js · 인프라

## 연락

글에 대한 의견이나 이야기 나누고 싶은 게 있다면 언제든지 환영합니다.
